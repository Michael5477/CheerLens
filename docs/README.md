# CheerLens Website

This folder contains the static website files for CheerLens, hosted on GitHub Pages.

## Files

- `index.html` - Main landing page
- `styles.css` - Stylesheet
- `privacy.html` - Privacy policy page
- `support.html` - Support and FAQ page

## GitHub Pages Setup

1. Go to repository Settings → Pages
2. Select source: Branch `main` → Folder `/website`
3. Save

## Custom Domain Setup

After GitHub Pages is enabled:
1. Add DNS records in GoDaddy (see main README)
2. Add custom domain in GitHub Pages settings: `cheerlens.com`
3. Enable "Enforce HTTPS"
