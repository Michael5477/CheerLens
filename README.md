# CheerLens - Smile Detection App

A professional iOS app for interview preparation and smile confidence training. Practice your interview skills with AI-powered facial expression analysis.

## Features

- **Smile Training**: Practice answering interview questions while the app analyzes your smile performance in real-time
- **Smile Monitoring**: Monitor your smile expressions without questions, focusing purely on facial expressions
- **Performance Analytics**: Detailed insights and charts showing your smile ratio and performance trends
- **Offline Functionality**: All features work completely offline - no internet required
- **One-Time Purchase**: Simple pricing model - $29.99 for unlimited access (removes daily 3-minute limit)

## Free Plan

- 3 minutes of total usage time per day (combined for both Smile Training and Smile Monitor)
- Access to all features with daily time limit
- Daily limit resets every 24 hours

## Unlimited Access

- One-time purchase: $29.99
- Removes daily time limit permanently
- All features remain the same - only the time restriction is removed

## Technical Details

- **Platform**: iOS 14.0+
- **Framework**: SwiftUI
- **ML Framework**: Google ML Kit Face Detection
- **In-App Purchase**: StoreKit 2 (Non-Consumable)

## Privacy

- All processing is done locally on your device
- No data is uploaded to any server
- Completely offline functionality
- No personal data collection

## Requirements

- iOS 14.0 or later
- Device with front-facing camera
- Camera permission required
